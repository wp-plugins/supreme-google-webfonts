== Description ==

This plugin simply activates, and makes accessible the ENTIRE Google Webfonts repository.  Your visual editor will show a new, third row with an easy drop-down list for choosing fonts.

I also added a box for font size selection... they seem to go hand in hand.

== Installation ==

1. Upload the plugin to your 'wp-content/plugins' directory, or download and install automatically through your admin panel.

     
2. Activate the plugin through the 'Plugins' menu in WordPress.

3. That's it.  Go to your visual editor to begin using your google webfonts.

4. If you are updating; no problem.  Just click the automatic update and everything will be done for you.
